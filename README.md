# Ansible Installation

This Ansible project intend to automate the installation of your fresh new Windows.

Using:

* Chocolatey
* Ansible

## Installation

Run: `Install.ps1`

## Core Apps

* Teams
* Slack
* Python
* VSCode
* SublimeText
* Skype
* Skype for Business
* Outlook
* Notepad++
* Linux subsystem (Debian)
* cfn-lint
* Dropbox
* Chrome
* Firefox
* FoxitPDF

## Windows Settings

* Natural Scroling for trackpad
* Folders List View
* Show file extension
* Show hidden files
