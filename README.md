# Ansible Windows Installation (AWI)

This Ansible project intend to automate the post-installation of your a fresh new Windows for SysOps / Developper Profiles

Using:

* Chocolatey
* Ansible

## Tested on

* Windows 10 (build xxx)
* Windows 8.1 (build yyy)

## Installation

Run: `.\install.ps1`
